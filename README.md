# Tubes DPBO: POS Warung Bu Wiwi

Aplikasi desktop Point of Sale (POS) sederhana berbasis Java Swing dan SQLite. Dibuat untuk memenuhi Tugas Besar mata kuliah Desain dan Pemrograman Berorientasi Objek (DPBO).

## Prasyarat
- Java Development Kit (JDK) 8 ke atas.
- Maven (Terintegrasi otomatis di VS Code / IntelliJ).

## Cara Menjalankan Aplikasi
1. Buka folder project ini di VS Code atau IntelliJ.
2. Tunggu hingga Maven selesai mengunduh dependensi (`sqlite-jdbc` dan `flatlaf`).
3. Cari dan jalankan file `src/main/java/pos/Main.java`.
4. Login dengan akun default berikut:
   - **Username:** `admin`
   - **Password:** `admin`

## Fitur
- **Kelola Barang:** CRUD (Tambah, Ubah, Hapus) stok barang.
- **Kasir:** Pencatatan transaksi penjualan, potong stok otomatis, hitung kembalian, cetak struk digital.
